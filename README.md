# inventory_frontend

# To run build runner
- `dart run build_runner build`

# To generate icon
- `dart run flutter_launcher_icons`

# To build app for release
- `flutter build appbundle`

# To take screenshot of the screen
- `flutter screenshot`

## List flutter and run it
- `flutter emulators`
- `flutter emulators --launch <emulator id>`
- `flutter run`
